# 🚀 loc8r - MEAN Stack Learning Project

This repository contains a **MEAN Stack (MongoDB, Express.js, Angular, Node.js)** web application developed as part of my class learning. The project helped me understand full-stack web development concepts, including frontend development with Angular, backend API creation using Express.js and Node.js, and database integration with MongoDB.

> **Note:** This project was created for educational purposes as part of my coursework to gain hands-on experience with the MEAN Stack.

---

# 📌 Project Overview

The objective of this project was to learn how modern full-stack web applications are built using the MEAN Stack by implementing core concepts such as:

- Angular frontend development
- Express.js backend
- Node.js server-side programming
- MongoDB database integration
- Routing
- RESTful APIs
- MVC Architecture

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/Mahesh-kodari/loc8r.git
```

## Navigate to the project

```bash
cd loc8r
```

## Install dependencies

```bash
npm install
```

## Run the application

```bash
npm start
```

Open your browser and visit:

```
http://localhost:3000
```
